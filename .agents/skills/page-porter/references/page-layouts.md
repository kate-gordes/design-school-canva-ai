# Page Layout Patterns

Skeleton layout patterns from monorepo `pages/anon_home/skeleton/skeleton.css` and shared layout conventions.

## Content container

```css
/* contentContainer — wraps page slots */
padding: 0 24px; /* default (mobile) */
@media (min-width: 600px) {
  padding: 0 48px;
}
@media (min-width: 1200px) {
  padding: 0 32px;
}
```

## Flush section (full-bleed)

```css
/* flushSection — breaks out of contentContainer padding */
margin: 0 -24px; /* default */
@media (min-width: 600px) {
  margin: 0 -48px;
}
@media (min-width: 1200px) {
  margin: 0 -32px;
}
```

## Containers

```css
/* container.xlarge — max-width with auto centering */
max-width: 1416px;
margin: 0 auto;

/* container.responsive */
max-width: 1288px; /* default */
@media (min-width: 1650px) {
  max-width: 1416px;
}
margin: 0 auto;
```

## Section spacers (between slots)

Two spacers are stacked between every pair of slots:

```tsx
// Primary spacer
<Spacer size={{ default: '8u', xLargeUp: '12u' }} />
// Secondary spacer (responsive vertical rhythm)
<Spacer size={{ default: '0', smallUp: '1u', mediumUp: '3u', largeUp: '4u', xLargeUp: '1u' }} />
```

Resolved values (`baseUnit = 8px`):

- `8u` = 64px, `12u` = 96px
- `1u` = 8px, `3u` = 24px, `4u` = 32px

## Footer Banner wrapper

```css
/* Full-bleed dark gradient banner (last slot) */
/* flushSection + dark ThemeBoundary */
background: linear-gradient(to right, #00c4cc, #5a32fa, #7d2ae8);
padding: 64px 0; /* mobile */
@media (min-width: 600px) {
  padding: 256px 0;
}
/* content max-width: 1416px (xlarge container) */
```

## Breakpoint reference

| Name       | Query                 | Value  |
| ---------- | --------------------- | ------ |
| `smallUp`  | `(min-width: 600px)`  | 600px  |
| `mediumUp` | `(min-width: 900px)`  | 900px  |
| `largeUp`  | `(min-width: 1200px)` | 1200px |
| `xLargeUp` | `(min-width: 1650px)` | 1650px |

## Page assembly (orchestrator responsibility)

The `page-porter` orchestrator assembles the page index file after all slots are ported:

```tsx
// src/pages/<PageName>/index.tsx — generated by orchestrator
import Header from './Header';
import HeroEditorial from './HERO_EDITORIAL';
import Showcases from './SHOWCASES';
// ... all slots in manifest order ...
import Footer from './Footer';

export default function AnonHome() {
  return (
    <>
      <Header />
      <div className={styles.contentContainer}>
        <HeroEditorial />
        <SectionSpacer />
        <Showcases />
        <SectionSpacer />
        {/* ... */}
      </div>
      <Footer />
    </>
  );
}
```
