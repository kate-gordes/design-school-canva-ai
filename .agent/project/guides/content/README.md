# Content Design Rules

Guidelines for writing text, content, and user-facing copy in the application.

## Files

### Core Content Guidelines
- **content_guide.md** - Canva's voice, tone, and writing principles
- **content_tasks.md** - Specific content patterns for buttons, navigation, forms, labels
- **error_patterns.md** - Error messages, alerts, and user feedback patterns

## Purpose

These rules ensure all user-facing text is:
- **Consistent** with Canva's voice and tone
- **Clear** and easy to understand
- **Helpful** and action-oriented
- **Accessible** and inclusive

## When to Use

Read these files before writing:
- Button labels
- Navigation text
- Form labels and placeholders
- Error messages and alerts
- Empty states
- Onboarding flows
- Help text and tooltips
- Success messages
