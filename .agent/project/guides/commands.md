## Commands Cheatsheet

Agents should not execute commands directly; propose them to the maintainer.

### Dev

```bash
npm run dev
```

### Build & Preview

```bash
npm run build
npm run preview
```

### Lint & Format

```bash
npm run lint
npm run format
npm run format:check
```

### Tests (Playwright)

```bash
npm run test:e2e
npm run test:e2e:headed
npm run test:e2e:ui
npm run test:e2e:visual
npm run test:e2e:ci
```
