# Background Remover (Landing Page)

## What This Is

Public marketing page showcasing Canva's AI-powered background removal tool to non-authenticated visitors.

## Purpose

Convert visitors into Canva users by demonstrating the value of AI background removal. This is a marketing/acquisition page, not the actual tool.

## User Journey

1. **Discovery** - User finds page through search, link, or main signed-out page
2. **Education** - User learns about AI background removal capabilities
3. **Visualization** - User sees example images and use cases
4. **Conversion** - User clicks "Try it now" → Prompted to sign up
5. **Activation** - After sign-up → User accesses actual background remover tool

## Value Proposition

**For Users:**

- Remove backgrounds instantly with AI
- No Photoshop skills required
- Professional results in seconds
- Perfect for e-commerce, social media, presentations

**Use Cases:**

- Product photography for online stores
- Professional headshots with clean backgrounds
- Social media graphics
- Marketing materials

## Key Messages

- **Speed**: "Remove backgrounds instantly"
- **Quality**: "Professional results"
- **Ease**: "No editing skills required"
- **AI-Powered**: Emphasizes cutting-edge technology

## Conversion Strategy

**Primary CTA**: "Try it now - Sign up free"

- Action: Navigate to home/sign-up page
- Removes friction by emphasizing "free"

**Secondary CTA**: "Back to tools"

- Allows exploration of other features
- Keeps user engaged in discovery

## Visual Design

- **Color Theme**: Purple (`#E5D4F5`) - Premium, AI-focused
- **Hero Banner**: Large, prominent feature showcase
- **Breadcrumbs**: Easy navigation back to main page
- **Clean Layout**: Minimal distractions, focus on value prop
